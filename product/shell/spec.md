# Application Shell Specification

## Overview
A clean, modern sidebar navigation shell with collapsible sidebar, user menu, and responsive design.

## Navigation Structure
- Dashboard → Main dashboard view
- Settings → Application settings
- Profile → User profile management

## Layout Pattern
Sidebar navigation on the left with collapsible behavior. Main content area on the right with a top bar for mobile navigation toggle.

## Responsive Behavior
- **Desktop:** Full sidebar visible with labels and icons
- **Tablet:** Collapsible sidebar, can be toggled
- **Mobile:** Hidden sidebar with hamburger menu toggle

## Design Notes
- Uses stone color palette for neutral tones
- Supports light and dark mode
- Smooth transitions for sidebar collapse
